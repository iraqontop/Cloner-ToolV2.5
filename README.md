## ملخص
تم إنشاء هذا المشروع لتسهيل حياتك، فبدلاً من قضاء ساعات في محاولة جعل خادمك جميلاً قدر الإمكان، يمكنك ببساطة استنساخ خادم باستخدام هذه الأداة

**لمزيد من المعلومات:** [ServerDiscord](https://discord.gg/jWqyCXUsF3)
